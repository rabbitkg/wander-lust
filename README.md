# 🌍 Wanderlust

A full-stack Tourism Management web application where users can explore destinations, book trips, manage bookings, and share travel experiences. Built with modern web technologies for a smooth and responsive user experience.

## 🔗 Live Website

🌐 Client: https://wanderlust-murex-nu.vercel.app/
🌐 Server: [***]

---

## ✨ Features

- 🔐 Authentication & Authorization
  - Email/Password login system
  - Secure JWT authentication
  - Protected routes

- 🧭 Explore Travel Destinations
  - View all tourist destinations
  - Detailed destination information
  - Beautiful travel cards & galleries

- 📅 Booking System
  - Book destinations easily
  - Manage personal bookings
  - Booking history page

- ➕ Add & Manage Destinations
  - Add new travel spots
  - Update destination information
  - Delete destinations

- 🎨 Modern UI/UX
  - Fully responsive design
  - Mobile-friendly layout
  - Smooth animations and clean interface

- ⚡ Fast Performance
  - Optimized frontend
  - REST API integration
  - Dynamic data loading
